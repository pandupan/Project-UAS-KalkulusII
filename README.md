# 🧮 Project UAS Kalkulus II

Proyek yang dibuat untuk memenuhi **tugas besar Ujian Akhir Semester (UAS)** mata kuliah **Kalkulus II**.

**Dibuat oleh:** Pandu Pangestu & Irsalina Yumna

## ✨ Fitur

- **Kalkulator Matriks** — Hitung determinan, minor, kofaktor, dan adjoin
- **Ordo 2×2** — Operasi matriks ordo 2×2
- **Ordo 3×3** — Operasi matriks ordo 3×3
- **Riwayat Perhitungan** — Catat hasil perhitungan sebelumnya
- **Antarmuka Interaktif** — Mudah digunakan untuk pembelajaran

## 🛠️ Tech Stack

- **React.js** — Frontend library
- **JavaScript** — Bahasa utama
- **Material UI (MUI)** — UI component library
- **React Router** — Navigasi halaman
- **Netlify** — Deployment
- **KaTeX/React LaTeX** — Rendering rumus matematika

## 🚀 Cara Menjalankan

```bash
npm install
npm start
```
